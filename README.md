hello-storm
===========

## Goal
> To prepare a minimal project [storm]

## Prerequests (choice one of them)
1. run it on the **[box]**
2. **or** install: **[java]**, **[mvn]** (and **[storm]** if you want test it more real life)

## Get it
```sh
git clone git://github.com/slon1024/hello-storm.git
cd hello-storm
```

##Run it
```sh
source run.sh
```
